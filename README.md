IRService
=========
IRService is a service that receive decoded infrared from serial and convert it into a media control command.  

## Features
* 遙控器控制播放音樂 停止播放 (用途：睡前已經關燈, 不需開電腦螢幕, 不需開手機螢幕, 太亮的螢幕傷眼睛)
